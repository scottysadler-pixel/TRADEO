# Trade1

**Trade1** is a **static, beginner-friendly FX toolkit** focused on **AUD/USD**. It has two parts:

| Part | What it is | How you open it |
|------|------------|-----------------|
| **Trade app** | Simple UI: two research-backed strategies (Chameleon + Catchup), position-size calculator, trade journal, and a **Data health** tab | `standalone/trade.html` or `standalone/index.html` (same app). Host the **`standalone/`** folder on [Vercel](https://vercel.com) (root directory = `standalone`) or open locally with a small HTTP server. |
| **Analyst trial** | Full preset backtester, replay, charts — generated by `npm run trial` | `standalone/trial_dashboard.html` (after a trial run) or `output/trial_dashboard.html` on your PC |

**Start here:** [`docs/USER_GUIDE.md`](docs/USER_GUIDE.md) — how to use the Trade app, refresh data, verify connectivity, and what “up to date” means.

**Developers / CLI pipeline:** [`docs/OPERATOR_GUIDE.md`](docs/OPERATOR_GUIDE.md)

## Quick start (Trade app on your PC)

1. Install [Node.js](https://nodejs.org/) and [Python3](https://www.python.org/) if you want to refresh data.
2. From this folder:
   ```powershell
   npm ci
   pip install -r scripts/requirements.txt
   pip install -r aud_strategy/requirements.txt
   npm run fetch:price
   python aud_strategy/scripts/fetch_real_data.py
   python scripts/update_chameleon.py
   python scripts/update_catchup.py
   ```
3. Serve `standalone` (JSON files must load over `http://`, not `file://`):
   ```powershell
   cd standalone
   python -m http.server 8080
   ```
4. Open `http://localhost:8080/` or `http://localhost:8080/trade.html`
5. Use the **Data health** tab to confirm JSON loaded and run a **live quote check** when online.

Double-click shortcuts (if present): **`Refresh Trade1 Data.cmd`**, **`Open Trading Apps.cmd`**, **`Deploy to Vercel.cmd`** — see [`docs/RUN_WITHOUT_CURSOR.md`](docs/RUN_WITHOUT_CURSOR.md).
